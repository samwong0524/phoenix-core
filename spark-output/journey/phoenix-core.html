⚠️ 流失风险 #1 — Onboarding
首次使用无任何引导（audit-1），用户面对 3 栏布局 + 15 个信息区完全不知所措。@skill 语法需偶然发现（audit-19）。建议：实现 react-joyride 引导 + 空状态说明。
⚠️ 流失风险 #2 — Engagement
Workflow 编辑器无撤销（audit-3），节点 Delete 无确认（audit-4），SSE 断连无重连（audit-27）。破坏性操作导致数据丢失。建议：全局 undo/redo + 操作确认 + SSE 指数退避重连。
★ 关键时刻 — Activation
用户首次在 Agent 群组中 @skill 发送任务并收到响应 + 拓扑图动画联动。这是"啊哈时刻"——多 Agent 协作价值首次感知。建议：确保此路径零摩擦，添加成就反馈。
