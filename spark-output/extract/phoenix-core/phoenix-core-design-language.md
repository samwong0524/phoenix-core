## 6. Motion & Animation

**Personality**: Corporate (Restrained, Trustworthy)
**Scene**: DASHBOARD

**Signature easing**: `cubic-bezier(0.2, 0, 0, 1)` (Material Design 3 Standard)
**Exit easing**: `cubic-bezier(0.4, 0, 1, 1)`

**Duration palette**:
| Name | Duration | Use |
| --- | --- | --- |
| Quick | 200ms | Button feedback, hover |
| Standard | 300ms | Cards, panels, messages |
| Slow | 400ms | Modals, page transitions |

**Framer Motion variants** (from `src/lib/motion.ts`):
- `buttonPress` — scale 1 → 0.97 → 1.02 → 1
- `fadeSlideUp` — y:8→0, opacity:0→1, 300ms
- `staggerContainer` — staggerChildren:30ms
- `graphNodeEntrance` — scale 0.85→1.05→1.0
- `modalOverlay` — opacity 0→1, 150ms/200ms
- `modalPanel` — y:16→0, 400ms in / y→10, 250ms out
- `sidebarSlide` — x:-16→0, 300ms in / 200ms out

**CSS keyframe animations**: grid-scroll, glitch, pulse-glow, msg-in, dotPulse, badgePulse, logoPulse, viz-reticle-pulse, streamPulse, cardFadeSlideUp, shimmer, toast-in, toast-out, qp-fade-in, qp-slide-in, qp-pulse, phx-spin

**Reduced motion**: `@media (prefers-reduced-motion: reduce)` strips all transforms, forces 0.01ms durations.

## 11-19. Quick Start Reference

### Adding a new component:
```tsx
"use client";
import { useState } from "react";
import { motion } from "framer-motion";
import { corporateVariants } from "@/lib/motion";

export function MyCard() {
  return (
    <motion.div
      variants={corporateVariants.fadeSlideUp}
      style={{
        background: "var(--bg-card)",
        border: "1px solid var(--border)",
        borderRadius: "var(--radius-md)",
        padding: "var(--space-4)",
        color: "var(--text-primary)",
      }}
    >
      Content
    </motion.div>
  );
}
```

### Color usage rules:
- Backgrounds: `--bg-void` → `--bg-panel` → `--bg-card` → `--bg-elevated` (layered depth)
- Text: `--text-primary` for headings, `--text-secondary` for body, `--text-dim` for captions
- Borders: `--border` default, `--border-bright` for interactive elements
- Accent: `--color-primary` for actions, `--color-secondary` for highlights
- Status: `--green`/`--yellow`/`--red` with their `-soft`/`-text` variants

### Responsive rules:
- Desktop ≥1024px: 3-column grid (220px sidebar + 1fr + 280px panel)
- Tablet 768-1023px: drawer sidebar + full-width chat + icon strip
- Mobile <768px: hamburger + full-width + bottom sheet
- Touch targets: ≥44px (tablet), ≥48px (mobile)
